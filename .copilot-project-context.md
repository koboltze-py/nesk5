# Nesk3 – Projektübersicht (Stand: v3.5.1 / 21.03.2026)
# Diese Datei dient als Gedächtnis für GitHub Copilot.

## Basics
- **App**: Nesk3 – Erste-Hilfe-Station-Verwaltung, DRK Flughafen Köln
- **Stack**: Python 3.13, PySide6, SQLite (lokal), Turso (Cloud-Sync)
- **Git Remote**: https://github.com/koboltze-py/nesk5.git
- **Branch**: main (aktuell `22750f2`)
- **EXE-Build**: PyInstaller via `Nesk3.spec` → Ausgabe in `G EXE/`
- **Build-Befehl**: `python3.13 -m PyInstaller Nesk3.spec --distpath "G EXE" --workpath build_tmp --noconfirm`
- **Backup-Pfad**: `C:\Daten\Backup Nesk3\` (ZIP-Archiv)

## GUI-Module (gui/)
- main_window.py – Hauptfenster mit Dashboard, Sidebar-Navigation, Fade-Animation (180ms)
- aufgaben.py – Aufgaben-Formular (Nachtschicht)
- aufgaben_tag.py – Aufgaben-Formular (Tagschicht)
- sonderaufgaben.py – Sonderaufgaben mit Dienstpläne-Tree, Ordner-öffnen, Wiederherstellen
- fahrzeuge.py – Fahrzeug-Verwaltung
- mitarbeiter.py – Mitarbeiter-Verwaltung (Sidebar: Kategorien + Sonder)
- mitarbeiter_dokumente.py – Dokumenten-Browser (6 Tabs + Ausdrucke + Krankmeldungen)
- telefonnummern.py – Telefonverzeichnis
- beschwerden.py – Beschwerden-Management
- dienstliches.py – Dienstliche Dokumente
- hilfe_dialog.py – Hilfe/Info-Dialog

## Design-Konventionen
- **Tab-Farbe einheitlich**: `#1565a8` (alle 11 GUI-Dateien)
- **Seitenübergänge**: QGraphicsOpacityEffect + QPropertyAnimation, 180ms OutCubic
- **Sidebar-Stil**: Dunkles Theme, weiße Schrift, Hover-Effekte
